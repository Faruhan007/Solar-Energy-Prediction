# 🌞 Solar Energy Production Prediction

## 📌 Overview
This project focuses on predicting solar energy production using Machine Learning techniques on a large-scale real-world dataset.

## 📊 Dataset
- Size: 218,000+ records  
- Features: Weather data, time-based variables, and environmental factors  

## ⚙️ Approach
- Data Cleaning & Preprocessing  
- Exploratory Data Analysis (EDA)  
- Feature Engineering (time & weather-based features)  
- Model Training using Random Forest Regressor  

## 📈 Results
- R² Score: **0.9999**  
- Mean Absolute Error (MAE): **46 kWh**  

## 🛠️ Tools & Technologies
- Python  
- Pandas, NumPy  
- Scikit-Learn  
- Matplotlib  

## 🚀 Outcome
Built a highly accurate model that can support solar energy planning, optimization, and investment decisions.
